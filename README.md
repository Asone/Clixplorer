# Clixplorer

Clique consensus (EIP-225) block explorer

More info at https://github.com/ethereum/EIPs/issues/225

## Compile

```npm run-script```

Dependency: browserify

## Issue

Buffer issue
After running ```npm install``` remove ```rm -r node_modules/browserify/node_modules/buffer```
